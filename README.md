# ◈ Laboration 3
Detta repository innehåller laboration 3 för kursen DT208G.

## ✦ Projektbeskrivning
Denna webbplats är skapad med hjälp av ramverket angular och har funktioner som att konvertera olika enheter mellan varandra.

## ⚙ Funktioner
* **Angular:**  Ramverket angular kommer med ett flertal nya funktioner som routing och data-binding för att minimera mängden kod som behövs skrivas.
* 
* **TypeScript:** TypeScript gör det möjligt att använda objektorienterad programmering. Kontroll av datatyper med mera sköts även av TypeScript då man får mycket fler och striktare varningar än med klassisk JavaScript.
* 
* **Komponenter:** Webbplatsen är uppbyggt av modulära komponenter dels för att snabbt kunna klippa och klistra sin kod men även för att bättra kodens läsbarhet.
